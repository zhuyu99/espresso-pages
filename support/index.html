Before you post

Please do not include private account, payment, device serial number, or personal data in public GitHub issues.

Menu bar

If Espresso is not visible, check the macOS menu bar near the system status items and any hidden menu bar area.

Launch at login

Open Espresso settings, toggle launch at login off, then turn it on again to refresh the macOS login item.

Notifications

Check macOS System Settings > Notifications > Espresso if session-end notifications do not appear.

Common checks

What to check first.

  • Confirm Espresso is running from the menu bar, not as a regular Dock app.
  • Start a timed session and watch for the visible countdown before leaving a long task.
  • Review notification permission if you expect a local alert when the countdown ends.
  • If your Mac sleeps after lock, logout, or shutdown, that can be normal macOS policy.
